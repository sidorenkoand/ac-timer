# ActiveCollab timer

Cross-platform timer for ActiveCollab (Linux, Windows, maybe MacOS)

## Getting Started

1. Clone the repository
1. Install npm dependencies "npm install"
1. Run app "npm run start"

### Pack

Run "npm run pack-linux" or "npm run pack-win" 

## Built With
